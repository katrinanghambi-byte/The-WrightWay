# the-wright-way-finance-app
Slogan: Save right, The Wright Way

  ## Our Story
The Wright Way is a company that provides financial onboarding, investment, and trading services for newly independent adults, post-graduates to be specific with benefits to up to 3 years post-graduation. The Wright Way manages user accounts, investment portfolios, automated contributions, educational materials, and trading execution. The system supports account administration, custody of cash and securities, transaction logging, portfolio valuation just to name a few.
  ## Our Roles
	Admin: Full access to manage products, securities, users, and view system logs.
	Manager: To create user accounts, propose trades, and view reports.
	User: To be able to view their own accounts, request deposits/withdrawals, review portfolio.
	System: Automated contributions, dividends ingestion, valuation updates.

  ## Our Goals
	Manage user profiles and data.
	Track user accounts i.e. cash and investment accounts.
	Record securities such as stocks, bonds, ETFs, market positions, and historical transactions.
	Support deposits/withdrawals, buys/sells, dividends, fees.
	Provide portfolio-level performance and current valuations.
